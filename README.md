Online Examination System

A simple online exam portal built using Node.js, Express, SQLite, HTML, CSS.

*Features

✔ User Signup & Login
✔ Take Exam
✔ Submit answers
✔ Auto score calculation
✔ Clean UI

*How to Run Locally

npm install
node server.js

Open:

[http://localhost:3000](https://online-exam-mv67.onrender.com)
